# Visual Basic methods to control DraftSight.
 Set of functions to execute DraftSight operations from MS Excel Environment combined with data stored in spreadsheet.
 
 ## Usage
 To use the code, simply download zip file/clone repository and import entire project into MS Excel Visual Basic Editor.
